# Breast Cancer Classification using KNN 

## Overview
This machine learning project classifies breast cancer tumors as:
- Malignant
- Benign

## Algorithm Used
- K-Nearest Neighbors (KNN)

## Dataset
Wisconsin Breast Cancer Dataset from Scikit-learn

## Features
- Data preprocessing
- Model training
- Accuracy evaluation
- Classification report
- Confusion matrix
- Visualization

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Outcome
Built a predictive model for cancer diagnosis with performance evaluation.
